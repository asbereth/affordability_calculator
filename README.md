# Australian Mortgage Affordability Calculator

A dependency-free browser calculator for estimating borrowing capacity from an affordable
principal-and-interest repayment.

## Run

```bash
npm start
```

Then open `http://localhost:4173`.

## Test

```bash
npm test
```

## Current Scope

- Repayment amount
- Annual interest rate
- Loan term
- Weekly, fortnightly, and monthly repayment frequencies
- Optional deposit
- Estimated loan amount
- Estimated property price
- Loan-to-value ratio
- Total interest over the term
- Repayment comparison for the estimated loan

The calculation is an estimate only. Australian lenders assess borrowing capacity using income,
expenses, debts, credit limits, buffers, and lender policy.
